Happy Campers

This is an app that allows users to search national parks and find camping sites to visit. We built it as a 3-member team. Technologies used were HTML, CSS, JavaScript, and RESTful API's.

Github

Meal Plan Magee

Meal Plan Magee (MPM) is a solo project using React. It allows users to randomly generate daily meals based on calories. Technologies used were JavaScript, CSS, React, RESTful API, and Material UI.

Github

Pocket Sandwich

This 3-member team project allows users to find eating options. It's catered to people with food restrictions. Users can find restaurants based on their location. Technologies used were JavaScript, EJS, CSS, Bootstrap, PostgresSQL, Express, and Sequelize.
